# QR-Code-Generator
This is a lightweight and responsive QR Code Generator built using HTML, CSS, and JavaScript. It allows users to quickly generate QR codes from any text or URL and displays them dynamically on the page.

Features

Instant QR Code Generation: Enter any text or URL and generate a QR code instantly.

Responsive Design: Works perfectly on desktop, tablet, and mobile devices.

Error Handling: Input field shakes and highlights in red if left empty, providing clear visual feedback.

Dynamic QR Display: QR code appears below the input when generated, with smooth animation.

Clean UI: Minimalistic design with modern styling using CSS.

Technologies Used

HTML5

CSS3 (including animations for shake/error)

JavaScript (DOM manipulation, dynamic QR generation)

Free QR Code API: https://api.qrserver.com/

Usage

Clone or download the repository.

Open index.html in any modern browser.

Enter text or URL in the input field.

Click “Generate QR Code” to see your QR code appear.
